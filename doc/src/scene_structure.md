Structure of the scene.json file
================================

```json
{
    "background": "ref_to_scene_background",
    "events":
        [
            "ref_event1",
            "ref_event2"
        ]
}
```

background
----------

Le nom du fichier background à charger pour cette scène spécifiquement.

events
------

Une liste des références d'évènements à tester.
